## Overview

This week deepens the understanding of MLP internals covered in Chapter 4. The report explains the mathematical foundations of neural networks: forward propagation, activation functions, loss functions, backward propagation, and the roles of dataset splitting in training.

## Key Concepts

- Single-layer vs. multi-layer perceptrons
- Activation functions: Sigmoid, ReLU, Softmax
- Forward propagation and backward propagation (gradient descent)
- Loss functions: MSE, Cross-Entropy
- Dataset splitting: training, validation, test sets
- Batch processing and epoch-based training
